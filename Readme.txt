# World Cities Travel Guide

## Описание проекта

Этот проект — интерактивный путеводитель по четырём популярным городам мира: Токио, Алматы, Париж и Нью-Йорк. На каждой странице представлена информация о достопримечательностях, советы для путешественников и возможность оставить отзыв. Проект также поддерживает сохранение пользовательских настроек, таких как предпочтения по теме и языку, чтобы улучшить удобство пользования.

## Ключевые особенности

### 1. Аутентификация пользователя
- **Вход и выход**: Пользователь может войти в систему, введя своё имя пользователя, и выйти с помощью кнопки «Выйти».
- **Сохранение сеанса**: Имя пользователя сохраняется в `localStorage`, что позволяет продолжать сеанс после перезагрузки страницы.
- **Технологии**: Используется JavaScript и `localStorage` для хранения информации о сеансе.

### 2. Предпочтения пользователя
- **Светлая и тёмная темы**: Пользователь может переключать тему сайта между светлой и тёмной.
- **Сохранение темы**: Выбранная тема сохраняется в `localStorage` и автоматически применяется при повторном посещении сайта.
- **Технологии**: JavaScript и `localStorage` для сохранения пользовательских предпочтений.

### 3. Фильтры и настройки
- **Язык цитаты**: Пользователь может выбрать язык цитаты, и выбранный язык сохраняется в `localStorage`.
- **Сохранение выбора языка**: Предпочтения пользователя по языку цитаты сохраняются и применяются при повторном посещении.
- **Технологии**: JavaScript и `localStorage`.

## Дизайн и типографика
Проект использует адаптивный дизайн с CSS, обеспечивая удобный интерфейс на всех устройствах. Типографика и цветовая схема выбраны для улучшения визуального восприятия и удобства чтения.

## Используемые технологии
- **HTML**: Разметка страниц и структура контента.
- **CSS**: Оформление, стилизация и адаптивный дизайн.
- **JavaScript**: Реализация интерактивных функций и работы с `localStorage`.
- **Bootstrap**: Используется для удобной стилизации и адаптивного поведения элементов.

## Запуск проекта
1. Клонируйте репозиторий.
2. Откройте файл `tokyo.html` в браузере для тестирования.

## Требования для проекта
1. Реализована аутентификация с использованием `localStorage`.
2. Настройки светлой/тёмной темы сохраняются.
3. Фильтры сохраняются в `localStorage` и восстанавливаются при загрузке страницы.
